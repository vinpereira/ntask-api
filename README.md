# NTask API

A Nodejs API, more for learning and fun with Express.

## Nodejs version and libraries
**Nodejs v8.9.4**

- bcrypt - 1.0.3
- body-parser - 1.18.2
- consign - 0.1.6
- express - 4.16.2
- jwt-simple - 0.5.1
- passport - 0.4.0
- passport-jwt - 3.0.1
- sequelize - 4.33.3
- sqlite3 - 3.1.13

### How to execute
1. Clone this repo
2. Install all dependencies -- `npm install` -- in project's root folder
3. Execute `npm start`
